贪吃蛇
Programmer jrw&wzy

请通过“贪吃蛇.bat”脚本进入游戏！！！不要通过src\main.exe！！！

进入游戏时请首先确定游戏是否全屏，若没有全屏，请手动全屏
进入游戏界面，使用上下键控制光标进行选择，回车键确认，选项如下：
1.Start New Game : 开始新游戏
2.View or Change Game Config and Map : 查看游戏配置与地图信息
3.Build New Config : 新建游戏配置
4.Draw New Map : 新建地图
5.View Record : 观看记录
6.Quit : 退出游戏

1.开始新游戏
你将进入新的贪吃蛇游戏，游戏中有三种分值的食物，食物会自动刷新，游戏中可以通过空格键暂停，用q键随时退出，游戏结束后会提示是否保存，按b保存，其余任意键退出

2.查看游戏配置与地图信息
该信息继承上一次编辑，选择Quit时退回游戏主界面

3.新建游戏配置
按照提示输入，任何分值食物概率不能为1，但可以为0，文件总数不能超过40，除输入文件外按q随时退出，通过提示确定是否保存

4.新建地图
按照提示输入，编辑长宽时使用p加m减，进入任意选项后都不能通过q直接退出，障碍物直接输入坐标，且障碍物总数不能超过15个，文件总数不能超过40

5.观看记录
请不要尝试在不同分辨率下查看记录！！观看中可以按q随时退出

6.退出游戏
直接结束程序
